## Benchmark Modalities

**Modality Types:** text, image, image_sequence

**Description:** Multi-modal benchmark combining text (context sentences with idiomatic expressions), static images (depicting literal/figurative meanings), and temporal image sequences (three-panel comic-style progressions)

## Analytical Tasks

**Task Types:** multimodal_ranking, image_completion, idiomaticity_classification

**Detailed Description:** Task A requires ranking images based on alignment with idiomatic vs literal meanings in context. Task B involves predicting narrative completion in image sequences while distinguishing literal from figurative interpretations. Both require deep semantic understanding of non-compositional language phenomena.

## Summary

AdMIRe introduces a multimodal benchmark for evaluating models' understanding of idiomatic expressions through image ranking and sequence completion tasks. The dataset contains 100 English and 55 Portuguese potentially idiomatic noun compounds with corresponding visual representations depicting both literal and figurative meanings.

## Key Findings

Top-performing systems achieved human-level performance (0.93 top-1 accuracy) using mixture-of-experts approaches with multiple LLM queries. However, this required substantial computational resources and prompt engineering. Models exhibited bias toward idiomatic interpretations regardless of context.
