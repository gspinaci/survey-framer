## Benchmark Modalities

**Modality Types:** text, image

**Description:** A multi-modal Visual Question Answering (VQA) benchmark combining digital images of fine art and cultural heritage with multiple-choice textual questions.

## Analytical Tasks

**Task Types:** visual question answering, instance identity, instance attribute, instance location, instance counting, spatial relation, instance interaction, visual-inspired reasoning

**Detailed Description:** Evaluates models on seven cognitive dimensions of art historical interpretation: Instance Identity, Instance Attribute, Instance Location, Instance Counting, Spatial Relation, Instance Interaction, and Visual-Inspired Reasoning (commonsense or causal inference).

## Summary

VQArt-Bench is a large-scale visual question answering benchmark designed to evaluate multimodal LLMs on the complex semantic interpretations required for visual art analysis. The dataset is built using a multi-agent pipeline to generate 14,463 nuanced, multiple-choice questions derived from Wikipedia descriptions of cultural heritage images. Extensive evaluation reveals that while proprietary models lead, even top-performing MLLMs struggle significantly with basic instance counting within complex artistic compositions.

## Key Findings

Proprietary models lead in artistic visual analysis, with Gemini 2.5 achieving the highest overall accuracy (71%), followed by the open-source Kimi-VL (67%). All models struggle significantly with art-focused instance counting (e.g., Gemini 2.5 scores 66%, LLaVA-NeXT at 31%) while performing comparatively better on higher-level visual-inspired reasoning tasks.
