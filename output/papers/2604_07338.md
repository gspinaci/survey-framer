## Benchmark Modalities

**Modality Types:** text, image

**Description:** Vision-language evaluation where models analyze cultural heritage object images and predict structured metadata fields (culture, period, origin, creator) which are then evaluated using LLM-as-Judge framework

## Analytical Tasks

**Task Types:** structured metadata inference, cultural attribution, historical period classification, creator identification, cross-cultural reasoning

**Detailed Description:** Models must infer non-observable cultural attributes from visual input alone, including cultural classification (e.g., Chinese, Greek), historical periodization (e.g., Tang dynasty, 5th century BCE), geographical origin, and creator attribution - requiring deep cultural and historical knowledge beyond visual perception

## Summary

Appear2Meaning introduces a cross-cultural benchmark for evaluating vision-language models on structured cultural metadata inference from heritage images, covering 750 objects across 4 cultural regions and 4 object categories. The benchmark evaluates whether VLMs can infer non-observable cultural attributes (creator, origin, period, culture) from visual input alone using an LLM-as-Judge framework.

## Key Findings

All models achieve very low exact match accuracy (0.005-0.029) but substantially higher partial match rates (0.453-0.658), indicating models capture fragmented signals but fail at coherent multi-attribute prediction. Performance varies significantly across cultures, with East Asia showing highest accuracy and Americas showing lowest. Models perform better on title and creator attributes while culture, period, and origin remain more challenging.
