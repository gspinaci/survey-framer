## Benchmark Modalities

**Modality Types:** text, image

**Description:** Multimodal benchmark where VLMs process heritage artifact images and generate structured cultural metadata predictions (culture, period, origin, creator) which are then evaluated using an LLM-as-Judge framework

## Analytical Tasks

**Task Types:** structured metadata inference, cultural attribution, temporal classification, creator identification, cross-cultural analysis

**Detailed Description:** Tasks require inferring non-observable cultural attributes from visual input including cultural classification (e.g., Chinese, Greek), historical period dating (e.g., Tang dynasty, 5th century BCE), geographical origin identification, and creator attribution - all requiring cultural knowledge beyond visual perception

## Summary

Appear2Meaning introduces a cross-cultural benchmark for evaluating vision-language models' ability to infer structured cultural metadata (creator, origin, period, culture) from heritage artifact images alone. The benchmark includes 750 objects across four cultural regions and four object categories, revealing that while models can capture partial cultural signals, they struggle with complete metadata inference and exhibit substantial performance variation across cultures.

## Key Findings

All evaluated VLMs achieved very low exact match accuracy (0.005-0.029) but higher partial match rates (0.453-0.658). Performance varied significantly across cultural regions, with East Asia showing highest accuracy and Americas showing lowest. Models performed better on title and creator attributes while struggling with culture, period, and origin inference.
