## Benchmark Modalities

**Modality Types:** text, image

**Description:** Multi-modal evaluation combining images with textual questions in both standard format (separate text and images) and vision-only format (questions embedded within screenshots/photos)

## Analytical Tasks

**Task Types:** multimodal reasoning, visual-textual integration, domain-specific knowledge application, complex problem solving

**Detailed Description:** College-level questions requiring integration of visual and textual information across multiple disciplines, with emphasis on preventing shortcut exploitation and testing true multimodal understanding rather than simple factual recall

## Summary

MMMU-Pro introduces a more robust version of the MMMU benchmark that filters out text-only answerable questions, increases candidate options from 4 to 10, and adds a vision-only input setting where questions are embedded in images. The benchmark aims to prevent shortcut exploitation and test true multimodal understanding across college-level questions in six disciplines.

## Key Findings

All tested models showed significant performance drops on MMMU-Pro compared to original MMMU, ranging from 16.8% to 26.9% decrease. GPT-4o achieved 51.9% average performance (54.0% standard, 49.7% vision-only). Chain-of-thought prompting generally improved performance, while OCR prompts had minimal effect. The vision-only setting particularly challenged models' ability to integrate visual and textual information.
